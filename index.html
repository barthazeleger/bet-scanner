🎯 Bet Scanner — hoe het werkt
Data-gedreven sportsbetting · versie 3.2.0 · april 2026
🤖

Bet Scanner combineert drie databronnen met een wiskundig model om dagelijkse value bets te vinden. Het doel: systematisch edge vinden waar bookmakers het mis hebben, en via Kelly Criterion slim inzetten.

🧠 Het model uitgelegd
1. No-vig kans berekening
Alle bookmaker odds worden genormaliseerd (marge verwijderd). Het gemiddelde over alle books = consensuskans.
2. Data-aanpassingen
Op basis van api-sports.io + ESPN wordt de kans bijgesteld:
  • Vorm laatste 5 wedstrijden (gewicht 35%)
  • H2H record (gewicht 22%)
  • Competitiestand / druk (gewicht 12%)
  • Thuisvoordeel per sport (5–6%)
  • Sleutelblessures (negatief, gewicht 8%)
  • Momentum laatste 3 vs. vorige 3 (gewicht 10%)
  • Inzet van de wedstrijd (degradatie / titelrace)
3. Kelly Criterion
k = (ep×(odd-1) − (1−ep)) / (odd-1)
Alleen picks met k > 1.5% worden geselecteerd. Units schalen met k.
4. Calibratie (zelflerend)
Na elke afgeronde bet past het model zijn drempel per markttype aan. 8+ bets per categorie nodig voor recalibratie.
📡 Databronnen
⚽
The Odds API
Live odds van 40+ bookmakers per wedstrijd.
500 calls/maand gratis · ~21 calls per scan · max ~23 scans/maand
📊
api-sports.io
Teamstatistieken, H2H, blessures, scheidsrechters, standen.
100 calls/dag gratis · ~45 calls per prematch scan
📺
ESPN Scoreboard API
Live scores + standen voor 17+ competities.
Volledig gratis, geen key nodig
🚀
Sportmonks (toekomst)
Echte xG, line-ups, diepere stats. Aanbevolen bij ROI > 10% over 20+ bets.
💰 Inzetstrategie
0.3U
Kelly < 4% · €3 · voorzichtig
0.5U
Kelly 4–8% · €5 · normaal
1.0U
Kelly 8–18% · €10 · sterk
2.0U
Kelly > 18% · €20 · uitzonderlijk
Startkapitaal €100 · 1 unit = €10 · max 5 picks/dag · min odds 1.60 · min edge 5.5% · min trefkans 28%
